## Inspección y Especificaciones

### Measure

**Propósito:** Medir distancias y espaciados visualmente.

- Muestra medidas entre elementos seleccionados
- Genera líneas de cota automáticas
- Útil para documentar espaciados en el diseño

**Uso en handoff:** Crear anotaciones visuales de espaciado para developers.

[Ver en Figma Community](https://www.figma.com/community/plugin/739918456607459153)

### Redlines

**Propósito:** Generar especificaciones técnicas automáticas.

- Crea anotaciones de dimensiones
- Muestra propiedades CSS
- Genera documentación visual

**Uso en handoff:** Documentar specs sin usar Dev Mode.

[Ver en Figma Community](https://www.figma.com/community/plugin/781354942292031141)

## Exportación de Código

### Anima

**Propósito:** Convertir diseños de Figma a código funcional.

- Exporta a HTML/CSS, React, Vue
- Mantiene Auto Layout como Flexbox
- Exporta animaciones y transiciones
- Genera código responsive

**Uso en handoff:** Proporcionar código base a developers.

[Ver en Figma Community](https://www.figma.com/community/plugin/857346721138427857)

### Figma to Code (HTML, Tailwind, Flutter, SwiftUI)

**Propósito:** Generar código en múltiples frameworks.

- Soporta: HTML/CSS, Tailwind CSS, Flutter, SwiftUI
- Convierte Auto Layout a Flexbox
- Respeta variables de diseño

**Uso en handoff:** Acelerar la implementación inicial.

[Ver en Figma Community](https://www.figma.com/community/plugin/842128343887142055)

### Locofy

**Propósito:** Convertir diseños a código production-ready.

- React, React Native, HTML/CSS, Vue, Angular
- Mantiene responsividad
- Genera componentes reutilizables
- Integración con GitHub

**Uso en handoff:** Generar código listo para producción.

[Ver en Figma Community](https://www.figma.com/community/plugin/1056467900248561542)

## Design Tokens

### Tokens Studio for Figma

**Propósito:** Gestionar y sincronizar design tokens.

- Define tokens de color, tipografía, espaciado
- Sincroniza con repositorios (GitHub, GitLab)
- Exporta a JSON, CSS, SCSS
- Soporta tokens semánticos y alias
- Integración con Style Dictionary

**Uso en handoff:** Mantener tokens sincronizados entre diseño y código.

[Ver en Figma Community](https://www.figma.com/community/plugin/843461159747178978)

### Design Tokens

**Propósito:** Exportar estilos como tokens.

- Exporta colores, tipografía, efectos
- Formatos: JSON, CSS, SCSS, XML (Android)
- Compatible con Style Dictionary

**Uso en handoff:** Generar archivos de tokens para el equipo de desarrollo.

[Ver en Figma Community](https://www.figma.com/community/plugin/888356646278934516)

## Documentación

### Figma Autoname

**Propósito:** Renombrar capas automáticamente usando IA.

- Detecta el tipo de elemento
- Genera nombres descriptivos
- Procesa múltiples capas a la vez

**Uso en handoff:** Limpiar nombres antes de entregar.

[Ver en Figma Community](https://www.figma.com/community/plugin/1160642826057169962)

### Notepad

**Propósito:** Agregar notas y documentación dentro de Figma.

- Notas tipo sticky note
- Markdown support
- Categorización por colores

**Uso en handoff:** Documentar decisiones y comportamientos.

[Ver en Figma Community](https://www.figma.com/community/plugin/739527655520996498)

### Component Documentation

**Propósito:** Documentar componentes sistemáticamente.

- Genera documentación de variantes
- Lista props y estados
- Crea páginas de documentación

**Uso en handoff:** Crear guías de componentes para developers.

## Assets y Optimización

### Export Kit

**Propósito:** Exportación masiva de assets.

- Exporta múltiples formatos simultáneamente
- Configura prefijos y sufijos
- Organiza en carpetas automáticamente

**Uso en handoff:** Preparar assets para desarrollo.

[Ver en Figma Community](https://www.figma.com/community/plugin/930069739294546982)

### SVGO Compressor

**Propósito:** Optimizar SVGs exportados.

- Reduce tamaño de archivos SVG
- Elimina metadata innecesaria
- Mantiene calidad visual

**Uso en handoff:** Entregar SVGs optimizados.

[Ver en Figma Community](https://www.figma.com/community/plugin/782713260363070260)

### TinyImage Compressor

**Propósito:** Comprimir imágenes PNG y JPG.

- Reduce peso sin pérdida visible
- Procesa múltiples imágenes
- Integración con TinyPNG API

**Uso en handoff:** Optimizar imágenes antes de entregar.

[Ver en Figma Community](https://www.figma.com/community/plugin/789009980664807964)

### Advanced SVG Export

**Propósito:** Control avanzado de exportación SVG.

- Exporta con IDs personalizados
- Control de viewBox
- Optimización automática

**Uso en handoff:** SVGs listos para implementación.

## Accesibilidad

### Stark

**Propósito:** Suite completa de accesibilidad.

- Verificador de contraste WCAG
- Simulador de daltonismo
- Verificador de tamaño de texto
- Generador de reportes

**Uso en handoff:** Validar accesibilidad antes de entregar.

[Ver en Figma Community](https://www.figma.com/community/plugin/732603254453395948)

### A11y - Color Contrast Checker

**Propósito:** Verificar contraste de colores.

- Verifica ratios WCAG AA y AAA
- Sugiere colores alternativos
- Verifica múltiples pares de colores

**Uso en handoff:** Asegurar que los colores cumplen estándares.

[Ver en Figma Community](https://www.figma.com/community/plugin/733159460536249875)

### Able

**Propósito:** Verificación de accesibilidad simplificada.

- Contraste de color en tiempo real
- Simulación de visión reducida
- Reportes de compliance

**Uso en handoff:** Quick check de accesibilidad.

[Ver en Figma Community](https://www.figma.com/community/plugin/734693888346260052)

### Focus Order

**Propósito:** Definir orden de tabulación.

- Establece orden de focus para screen readers
- Visualiza el flujo de navegación
- Genera anotaciones de orden

**Uso en handoff:** Documentar orden de navegación por teclado.

## Organización y Limpieza

### Clean Document

**Propósito:** Limpiar archivos antes del handoff.

- Elimina estilos no usados
- Remueve capas ocultas
- Limpia grupos vacíos
- Detecta elementos fuera del frame

**Uso en handoff:** Preparar archivo limpio para entregar.

[Ver en Figma Community](https://www.figma.com/community/plugin/767379019764649932)

### Sorter

**Propósito:** Ordenar capas automáticamente.

- Ordena alfabéticamente
- Ordena por posición (top-to-bottom, left-to-right)
- Ordena por tipo de elemento

**Uso en handoff:** Organizar capas de forma lógica.

[Ver en Figma Community](https://www.figma.com/community/plugin/742038190980789811)

### Instance Finder

**Propósito:** Encontrar todas las instancias de un componente.

- Localiza dónde se usa cada componente
- Muestra instancias con overrides
- Ayuda a auditar uso de componentes

**Uso en handoff:** Verificar consistencia de componentes.

[Ver en Figma Community](https://www.figma.com/community/plugin/741895659787979282)

### Unused Components

**Propósito:** Detectar componentes sin usar.

- Lista componentes huérfanos
- Identifica componentes duplicados
- Ayuda a limpiar la biblioteca

**Uso en handoff:** Eliminar componentes innecesarios.

### Rename It

**Propósito:** Renombrar capas en masa.

- Find & Replace en nombres
- Agregar prefijos/sufijos
- Numerar secuencialmente
- Usar regex

**Uso en handoff:** Estandarizar nomenclatura rápidamente.

[Ver en Figma Community](https://www.figma.com/community/plugin/731271836271143349)

## Colaboración

### Loom

**Propósito:** Grabar videos explicativos.

- Graba pantalla + cámara
- Embed directo en Figma
- Ideal para explicar interacciones

**Uso en handoff:** Crear walkthroughs de diseño.

[Ver en Figma Community](https://www.figma.com/community/plugin/1256592887498498754)

### Notion

**Propósito:** Conectar diseños con documentación.

- Embed frames de Figma en Notion
- Sincronización automática
- Contexto adicional para developers

**Uso en handoff:** Centralizar documentación.

### Jira

**Propósito:** Vincular diseños con tickets.

- Adjunta frames a issues
- Crea issues desde Figma
- Tracking de implementación

**Uso en handoff:** Conectar diseño con gestión de proyecto.

### Storybook Connect

**Propósito:** Vincular componentes Figma con Storybook.

- Conecta componentes de Figma con sus contrapartes en código
- Muestra estado de implementación
- Facilita comparación diseño vs código

**Uso en handoff:** Tracking de implementación de componentes.

[Ver en Figma Community](https://www.figma.com/community/plugin/1056265616080331589)

## Configuración Recomendada

### Kit Básico (Gratuito)

1. **Measure** - Documentar espaciados
2. **Clean Document** - Limpiar archivo
3. **Rename It** - Estandarizar nombres
4. **A11y Contrast Checker** - Verificar accesibilidad
5. **SVGO Compressor** - Optimizar SVGs

### Kit Profesional

Todo lo anterior más:

1. **Tokens Studio** - Gestión de tokens
2. **Stark** - Suite de accesibilidad
3. **Anima** - Exportación de código
4. **Storybook Connect** - Conexión con código

## Consideraciones

- Verifica la compatibilidad con tu plan de Figma
- Algunos plugins tienen versiones gratuitas limitadas
- Prueba los plugins en un archivo de test antes de usar en producción
- Mantén los plugins actualizados
